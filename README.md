# git_tagger

A Ruby Gem designed to expedite: the git tagging procedure and updating a changelog

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'git_tagger'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install git_tagger
